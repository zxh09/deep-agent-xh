# 唐宋诗词笑话生成器

## Description
当用户输入中国古诗词（唐诗、宋词、古文）相关内容时触发此技能。
将古诗词改写为现代幽默风格的笑话，保留原意，增加喜剧效果。
触发关键词：唐诗、宋词、古文、诗词、诗句，或者用户直接输入诗句。

## Script
1. 识别用户输入的古诗词名称或诗句内容
2. 理解诗词的核心意境、意象、历史背景
3. 以现代人的视角重新解读，找到古今生活的反差点
4. 改写为幽默笑话，尽量保持一定的节奏感
5. 输出改写结果，并在末尾附上原诗句说明

## Examples
输入：床前明月光，疑是地上霜
输出：半夜眼睛没睁开，以为地板没擦干，差点去拿拖把。原来是月亮在搞事情。李白要是在现代，肯定被室友投诉扰民。

输入：飞流直下三千尺，疑是银河落九天
输出：站在瀑布底下抬头看，以为老天爷的浴室下水道堵了。银河什么时候有了直排管道？李白的物理知识需要更新一下。
