# 技术文档生成器

## Description
当用户需要生成技术文档、README 文件、API 说明、使用手册时触发此技能。
触发场景：用户说「帮我写文档」「生成 README」「写接口说明」等。

## Script
1. 理解用户提供的代码结构或功能描述
2. 确定文档类型：
   - README：项目介绍、安装步骤、使用示例
   - API 文档：接口列表、参数说明、返回值、错误码
   - 使用手册：操作步骤、配置说明、常见问题
3. 生成标准 Markdown 格式文档，包含：
   - 清晰的标题层级
   - 代码示例（使用 ``` 代码块）
   - 表格（适合展示参数列表）
4. 如果用户指定了文件名，将文档写入对应文件
5. 文档语言与用户输入语言保持一致
